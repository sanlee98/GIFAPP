1.Used GIPY api to get gif and stickers data.

2.In home screen gif data is shown in grid layout.

3.In Stickers screen stickers are rendered.

4.On click of any item details page will open.

5.With the help of search input we can filter the rendering of gif and stickers.

6.Pagination with infinite scroll is enabled.
